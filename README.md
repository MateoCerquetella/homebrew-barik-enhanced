# Homebrew Tap for Barik Enhanced

Custom Menu Bar for macOS with 20+ configurable widgets.

## Install

```bash
brew tap MateoCerquetella/barik-enhanced
brew install --cask barik-enhanced
```

## Update

```bash
brew upgrade --cask barik-enhanced
```

## Uninstall

```bash
brew uninstall --cask barik-enhanced
```
